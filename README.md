SQL Portfolio - Sakila (Business Case Analysis)

This repository contains a business-oriented SQL analysis using the Sakila database.

👨‍💻 About Me
I am a Data Analyst with experience in Finance, HR, and Supply Chain analytics.  
I am currently strengthening my SQL skills through hands-on practice and real-world datasets.

📊 Project Objective
The goal of this project is to simulate real business scenarios using SQL by answering key performance questions.

This project focuses on:
- Revenue analysis
- Customer segmentation
- Product (movie) performance
- Business insights generation

🧠 Business Case
The analysis is based on a set of business questions provided by management, aiming to evaluate overall company performance.

Key questions include:
- Monthly revenue trends
- Top-performing categories
- Most rented movies
- High-value customers
- Revenue analysis by category (Action vs Comedy)

🗂 Repository Contents
- `basic-excercises.sql`: Contains SQL queries for business analysis
- `sakila.db`: Database used for the analysis

💡 Skills Demonstrated
- SQL for business analysis
- Aggregations and revenue calculations
- Joins across multiple tables
- Customer and product analysis
- Data-driven insights

🛠 Tools Used
- SQL (SQLite / MySQL depending on dataset)
- DB Browser / SQL environment
- GitHub
